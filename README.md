# Laboratorio 2 – Procesamiento de Datos con Python

Este laboratorio tiene como objetivo aplicar funciones avanzadas de Python 
como `lambda`, `map`, `filter` y `reduce` para procesar datasets y construir 
pipelines de procesamiento de datos.

## Contenido del repositorio

- `lab2.ipynb` : Notebook con el desarrollo de todas las actividades del laboratorio.

## Tecnologías utilizadas

- Python 3
- Jupyter Notebook
